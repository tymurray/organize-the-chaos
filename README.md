# Organize the Chaos

Personal weekly planning and task management system using markdown and git.

## Structure

- `weekly-plan.md` - Current week's plan with daily schedules, deadlines, and notes
- Version controlled via git for history and backup
- Synced to GitHub for access from anywhere

## Usage

Edit `weekly-plan.md` throughout the week to track tasks and updates. Commit and push changes regularly to maintain history.
